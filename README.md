## 🌟 Featured Projects

### ⛵ [VendeeGlobeRL](https://github.com/alessandrocampedelli/VendeeGlobeRL)
> **Autonomous Reinforcement Learning Yacht Navigator**
*   **Description:** A PPO Reinforcement Learning agent navigating a simulated IMOCA 60 sailboat around the world using real-world ERA5 wind data inside a custom Gymnasium environment.
*   **Technologies:** Python, Stable-Baselines3, Gymnasium, Pandas, NumPy, Jupyter Notebook.
*   **Why it's featured:** Showcases advanced Machine Learning concepts (Deep Reinforcement Learning), physical modeling, and preprocessing of real climate/meteorology datasets.

### 🎬 [CAST](https://github.com/alessandrocampedelli/CAST)
> **Cinematic Analysis & Screenplay Transformer**
*   **Description:** An automated NLP pipeline that parses screenplays from PDF/HTML, converts them to TEI-XML, and extracts location, environment, and temporal patterns across scenes.
*   **Technologies:** Python, Streamlit, NLTK, BeautifulSoup4, pdfminer, Plotly.
*   **Why it's featured:** Highlights a complete end-to-end data ingestion and analysis pipeline, NLP tokenization, digital humanities XML serialization, and interactive web dashboard development.

### 👁️ [retinal-superResolution](https://github.com/alessandrocampedelli/retinal-superResolution)
> **Medical Retina Image Super-Resolution**
*   **Description:** Enhancing resolution of medical retinal images using three distinct paradigms: optimization (FISTA + Wavelet $L_1$ regularization), deep learning (Hybrid Attention U-Net), and generative AI (DiffPIR diffusion models).
*   **Technologies:** Python, PyTorch, Jupyter Notebook, Google Colab.
*   **Why it's featured:** Demonstrates computer vision expertise in medical imaging, comparing classical optimization techniques with modern generative models.

### 📱 [TravelCompanion](https://github.com/alessandrocampedelli/TravelCompanion)
> **Native Android Travel Planner**
*   **Description:** A feature-rich native Android application designed to plan and document trips, with map integration, geolocation tracking, and offline data storage.
*   **Technologies:** Kotlin, Jetpack Compose, Room (SQLite), Dagger Hilt, Retrofit, Google Maps SDK, WorkManager.
*   **Why it's featured:** Represents modern mobile development best practices (MVVM, Dependency Injection, offline-first data synchronization, and UI building with Jetpack Compose).

### 🎓 [Selfie](https://github.com/alessandrocampedelli/Selfie)
> **Academic Life Manager**
*   **Description:** A web application helping university students coordinate schedules, tasks, and notes, featuring AI integrations for study assistance.
*   **Technologies:** TypeScript, Angular v19, Express, Node.js, MongoDB, Mongoose, OpenAI API.
*   **Why it's featured:** Proves fullstack development capabilities, NoSQL database design, state management, and standard web REST APIs with AI features.
